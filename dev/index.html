PlantGeom.Material — Type

Data structure for a mesh material that is used to describe the light components of a Phong reflection type model. All data is stored as RGBα for Red, Green, Blue and transparency.

PlantGeom.get_ref_meshes — Method
get_ref_meshes(mtg)

Get all reference meshes from an mtg, usually from an OPF.

Examples

using MultiScaleTreeGraph, PlantGeom
file = joinpath(dirname(dirname(pathof(MultiScaleTreeGraph))),"test","files","simple_OPF_shapes.opf")
opf = read_opf(file)
meshes = get_ref_meshes(opf)

using MeshViz
viz(meshes[0]["mesh"])
viz(meshes)

PlantGeom.meshes_color — Function

Find each mesh color.

Notes

meshes is typically the output from [meshes_from_topology].

Exemples

opf= readopf("simpleOPFshapes.opf") meshdata= meshesfromtopology(opf) shapes= extractopfshapes(opf) materials= extractopfmaterials(opf) cols= meshescolor(opf,meshdata) mesh(mergemeshes(meshdata["mesh"]), color= cols)
